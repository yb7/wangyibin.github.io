根据访问的user agent重定向网址

用例：当用户访问一个下载地址时，根据客户端类型

  • 如果是android手机，则直接掉转到apk下载
  • 如果是iPhone手机，则跳转定制的敬请期待页面
  • 如果是其他，则跳转到默认页面
server {
    listen 80;
    server_name app.chineseshopping.cn;
    charset utf-8;
    location / {
        root /home/xq/appsite;
    }

    location /d/ {
        alias /home/xq/ftp_root/download/;
    }
    location =/d {
        if ($http_user_agent ~* android) {
            return 301 http://app.chineseshopping.cn/d/android/OverSea.apk;
        }
        if ($http_user_agent ~* iphone) {
            return 301 http://app.chineseshopping.cn/download_iphone.html;
        }
        return 301 http://app.chineseshopping.cn/download_browser.html;
    }
}

通过 if 来进行user agent的判断，使用 ~* 来进行正则比较，*代表不区分大小写。
