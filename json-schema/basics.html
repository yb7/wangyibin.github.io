声明一个JSON Schema

JSON Schema本身也是一个JSON。

关键字$schema被用来声明该JSON为一个JSON Schema，但$schema并不是必须的。

{ "$schema": "http://json-schema.org/schema#" }

这里可以看到详细的解释。
