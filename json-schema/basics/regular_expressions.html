正则匹配

关键字pattern用于对数据值进行正则匹配。 正则表达式使用JavaScript ECMA中的定义。 但不完全支持。我们建议使用如下的关键字列表

案例如下

{
   "type": "string",
   "pattern": "^(\\([0-9]{3}\\))?[0-9]{3}-[0-9]{4}$"
}
  • "555-1212" 合规
  • "(888)555-1212" 合规
  • "(888)555-1212 ext. 532" 不合规
  • "(800)FLOWERS" 不合规
